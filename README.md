# Frosz — Gelato Fit

Demo conceptual para Frosz inspirada en una dirección visual de alto impacto: tipografía protagonista, bloques de color, producto flotante y transiciones entre sabores.

## Incluye

- Portada animada y adaptable a celular.
- Historia y atributos de la marca.
- Selector interactivo de nueve sabores.
- Puntos de venta filtrables por zona.
- Imágenes reales de Frosz y piezas de campaña creadas para la demo.
- Tarjeta social personalizada para compartir el enlace.

## Uso local

El sitio es estático. Se puede abrir `index.html` directamente o servir la carpeta con cualquier servidor web local.

## Créditos

Contenido de marca y fotografías de producto basados en [frosz.com.ar](https://frosz.com.ar/). Esta publicación es una demo conceptual.
